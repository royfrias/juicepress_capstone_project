## Juice Press Intranet App

## Team Project Created By Team JuicePress @UpRight Education FullStack SoftWare Development BootCamp

## Overview
The Juice Press Intranet App is a communication platform designed to streamline internal communication within Juice Press. This application provides a simple way for employees to interact and stay updated with company announcements and updates. Built by Team JuicePress as part of the UpRight Education FullStack Software Development BootCamp, this app features a robust announcement system accessible only to administrators, managers, and supervisors, ensuring that important information reaches the relevant personnel effectively.

## Features
Announcement System: Allows administrators, managers, and supervisors to make and edit announcements visible to the entire company. User Roles: Differentiates access levels, with regular employees having limited access compared to higher-ups. User-Friendly Interface: Designed for ease of use to ensure smooth communication across the company. Team
